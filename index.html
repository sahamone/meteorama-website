Weather Forecasting Reinvented

A Rust-based application providing easy access to weather forecasts using our own custom statistical model, combining the best of web technologies with Rust's robustness.

Key Features

Real-time Weather Data

Access current weather conditions anywhere in France with data from OpenWeatherMap

Advanced Forecasting

Unique statistical forecasting model for quality short- and medium-term predictions

Modern Architecture

Built with Tauri and React/Vite, combining web technologies with Rust's performance

Technology Stack

Frontend

  • React & TypeScript
  • TailwindCSS & DaisyUI
  • OpenLayers for dynamic maps
  • Lucide icons
  • Vite build tool

Backend

  • Rust with Tauri framework
  • Moka for cache management
  • Tokio for asynchronous requests
  • Reqwest for API requests
  • Serde for JSON handling

Data Sources

OpenWeatherMap API

Provides real-time weather data and forecasts
